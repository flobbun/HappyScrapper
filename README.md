# HappyScrapper ~ Google news web scrapper
<br><br>


<h2> INSTALLATION </h2>
<ul>
  ♦ Clone the repository
  <br>
  ♦ Open HappyScrappy.json
  <br>
  ♦ Fill the EMAIL_FROM, EMAIL_TO and EMAIL_PASSWORD fields and save
  <br>
  ♦ Use "pip install selenium"
  <br>
  ♦ Use "py HappyScrappy.py"
</ul>
  
<h2> DEPENDENCIES </h2>

<ul>
  ♦ Python 3
  <br>
  ♦ Edge browser
  <br>
  ♦ Selenium
</ul>
