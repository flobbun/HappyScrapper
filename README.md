# HappyScrapper
<h1>Google news web scrapper</h1>

<h2> INSTALLATION </h2>

<ul>
  ♦ Clone the repository
  <br>
  ♦ Open HappyScrappy.json
  <br>
  ♦ Fill the EMAIL_FROM, EMAIL_TO and EMAIL_PASSWORD fields and save
  <br>
  ♦ Use "pip install selenium"
  <br>
  ♦ Use "py HappyScrappy.py"
</ul>
  
<h2> DEPENDENCIES </h2>

♦ Python 3
♦ Edge browser
♦ Selenium
